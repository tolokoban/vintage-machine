# LOCATE

> LOCATE(colonne, ligne)

L'écran est composé de 30 lignes et 40 colonnes.
Cette fonction permet de définir où sera affiché le prochain caractère avec un [PRINT](PRINT).

Exemple :

```ts
CLS()
LOCATE(15,14)
PRINT("Coucou !")
```

voir aussi [CLS](CLS), [MOVE](MOVE), [PRINT](PRINT).
